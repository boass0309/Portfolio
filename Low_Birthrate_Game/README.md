# Low Birthrate Awareness Game

## Overview
This game is designed to raise awareness about the low birthrate issue in South Korea and its social implications. It aims to educate players about the causes and consequences of this issue through interactive gameplay.

## Technologies Used
- Unity
- C#

## Features
- Story-driven gameplay addressing social issues
- Interactive game mechanics
- Multiplayer support

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/vmffotltka/Portfolio.git
    cd portfolio/Low_Birthrate_Game
    ```
2. Open the project in Unity.
